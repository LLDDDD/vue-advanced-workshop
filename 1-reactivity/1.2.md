# 1.2 依赖跟踪

## 目标

- 新建一个包含`depend`方法和`notify`方法的`Dep`类
- 创建一个`autorun`函数，接收一个更新函数
- 在更新函数中，你可以调用`Dep`类的实例方法`dep.depend()`明确依赖关系
- 接着，你可以调用`dep.notify()`触发更新函数，让它重新执行。

完整的使用例子如下:

``` js
const dep = new Dep()

autorun(() => {
  dep.depend()
  console.log('updated')
})
// 打印: "updated"

dep.notify()
// 打印: "updated"
```

执行单元测试：

``` bash
npm test -- -t 1.2
```
